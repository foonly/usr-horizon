# System Variations

This lists changes to the system

## Traits

### Core Traits (Always Present)

- Fortitude
- Intelligence
- Initiative
- Willpower
- Awareness
- Mobility
- Melee
- Ranged

### Skill Traits (Settings dependent)

- Medicine
- Engineering
- Survival
- Subterfuge
- Presence
- Connect
- Artisan
- Administer
- Academics
- Piloting

## Specializations

### Core Trait Specializations

- **Mobility**: Stealth, Running, Climbing, Swimming, Acrobatics, Jumping
- **Melee**: Unarmed, Axes, Swords, Knives, Clubs, Bayonets, Brawling, Whips
- **Ranged**: Pistols, Rifles, Shotguns, Lasers, Bows, Throwing, Heavy Weapons

### Skill Trait Specializations

- **Medicine**: First Aid, Surgery, Treatment, Assessment, Pharmacology, Veterinary, Forensics, Psychiatry
- **Engineering**: Computers, Robotics, Cybernetics, Power Systems, Structural, Explosives
- **Survival**: Traps, Tracking, Herbology, Hunting, Fishing, Foraging, Scouting, Weather Sense, Hostile Environments
- **Subterfuge**: Lockpicking, Pickpocketing, Camouflage, Forgery, Disguise, Safe Cracking, Sleight of Hand, Shadowing
- **Presence**: Persuasion, Intimidation, Leadership, Entertainment, Diplomacy, Deception
- **Connect**: Streetwise, Trading, Underworld, Contacts
- **Artisan**: Fabrication, Electronics, Mechanical, Chemistry, Cooking
- **Administer**: Bureaucracy, Law, Accounting, Management
- **Academics**: History, Science, Xenobiology, Research, Languages
- **Piloting**: Starships, Atmospheric, Astrogation, Sensors
